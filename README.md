# Langton

Simulation of [Langton's ant](https://en.wikipedia.org/wiki/Langton%27s_ant).

<div align="center">
  <img src="assets/langton.gif" width=500>
</div>

## Usage

Requirements:

- [Poetry](https://python-poetry.org/)

Install the backend package

```bash
poetry install
langton run output.mp4
```
